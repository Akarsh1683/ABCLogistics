- Modern Logistics Management Platform

-- About the Project

This project is a logistics management system that I developed to handle basic operations like booking, managing deliveries, and tracking orders. The main idea behind this project is to simplify the logistics process and make it easier to manage data in one place.

-- Features

* User login and registration
* Booking system for logistics services
* Admin panel to manage users and bookings
* Basic tracking functionality

-- Technologies Used

* Java
* Spring Boot
* MySQL
* HTML, CSS, JavaScript

-- How to Run

1. Clone the repository
2. Import the project into your IDE (STS / Eclipse)
3. Configure the database in application.properties
4. Run the Spring Boot application

-- What I Learned

While working on this project, I learned how to:

* Build backend applications using Spring Boot
* Connect Java applications with MySQL
* Structure a full-stack project
* Handle basic CRUD operations
